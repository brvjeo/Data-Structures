## C++ Hierarchical Tree
```
Structure<Type>
├── LinkedList<Type>
│   ├── Stack<Type, LinkedList>
│   ├── Deque<Type, LinkedList>
│   └── Queue<Type, LinkedList>
└── Vector<Type>
    ├── Stack<Type, Vector>
    ├── Deque<Type, Vector>
    └── Queue<Type, Vector>
```
